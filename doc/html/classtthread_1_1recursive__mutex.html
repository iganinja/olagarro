Detailed Description

Recursive mutex class. This is a mutual exclusion object for synchronizing access to shared memory areas for several threads. The mutex is recursive (i.e. a thread may lock the mutex several times, as long as it unlocks the mutex the same number of times).

void tthread::recursive_mutex::lock ( )
inline

Lock the mutex. The method will block the calling thread until a lock on the mutex can be obtained. The mutex remains locked until unlock() is called.
