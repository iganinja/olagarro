void tthread::thread::detach ( )

Detach from the thread. After calling detach(), the thread object is no longer assicated with a thread of execution (i.e. it is not joinable). The thread continues execution without the calling thread blocking, and when the thread ends execution, any owned resources are released.

void tthread::thread::join ( )

Wait for the thread to finish (join execution flows). After calling join(), the thread object is no longer associated with a thread of execution (i.e. it is not joinable, and you may not join with it nor detach from it).
