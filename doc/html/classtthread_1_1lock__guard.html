Detailed Description

template<class T>
class tthread::lock_guard< T >

Lock guard class. The constructor locks the mutex, and the destructor unlocks the mutex, so the mutex will automatically be unlocked when the lock guard goes out of scope. Example usage:

mutex m;
int counter;
void increment()
{
lock_guard<mutex> guard(m);
++ counter;
}
